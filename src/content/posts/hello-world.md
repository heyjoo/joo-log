## 안녕하세요!

joo-log 블로그에 오신 것을 환영합니다. 이 블로그는 **Astro**와 **Tailwind CSS**를 사용하여 만들어졌습니다.

## 주요 기능

이 블로그는 다음과 같은 기능을 제공합니다:

- 마크다운 기반 게시글 작성
- 다크/라이트 모드 지원
- 반응형 디자인
- 코드 신택스 하이라이팅
- Obsidian 연동

## 코드 예시

다음은 간단한 JavaScript 코드 예시입니다:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to joo-log, ${name}`;
}

greet('World');
```

TypeScript도 지원합니다:

```typescript
interface Post {
  title: string;
  description: string;
  date: Date;
  tags: string[];
}

const post: Post = {
  title: 'Hello World',
  description: '첫 번째 게시글',
  date: new Date(),
  tags: ['블로그', 'astro'],
};
```
